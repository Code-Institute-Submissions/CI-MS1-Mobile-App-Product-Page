Features

Day view - without scrolling

Day view - without scrolling

You never loose overview of your day. Define the start and end time in the preferences to display our typical day.

Quick action

Quick action

Easy access buttons to change the alert/repeat option of an event or to delete an event

Privacy

Privacy

The app only needs permission to access your standard iOS calendar. No need for location, contacts, or other data on your iPhone.

New events

New events

Choose a start time and create an event with just enough options to keep it simple.

Edit events

Edit events

Make changes to a calendar event. Use a minimum of options to make changes, or drag the time in the event direclty.

Multiple calendars

Multiple calendars

View all events from all calendars. Quickly switch to your favourite calendar.
